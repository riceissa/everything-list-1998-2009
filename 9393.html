n^0 = 1, because 1= (n^m)/(n^m) = n^(m-m) = n^0.
Or better n^0 = the number of functions from the empty set (cardinal 0)
to the
set with cardinal n. This justifies also 0^0 = 1 (there is one (empty)
function from
the empty set to the empty set).

right?
