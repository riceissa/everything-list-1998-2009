Jacques Mallah wrote:
>(I said "seems to" because my example assumed there was only one set of
>physical laws. It didn't take into account other possibilities such as
>some of the stuff discussed on the everything-list. Thus, even if there is
>only one possible theory of quantum gravity, other types of physical laws
>could also exist as weighted by complexity. The point being that QM + SM +
>gravity is already too complex to be the only physical reality, and as much
>of an anthropic constraint as the parameters. The advantage will always
>belong to *some* type of ensemble theory, erasing the anthropic
>"coincidence".
>Ensemble #2 easily beats non-ensemble #1, but "apparent ensemble #2 with
>Tegmark ensemble" is not necessarily any more likely than such an "apparent
>non-ensemble #1 with Tegmark ensemble". Tricky! (And similar to the
>effect of the MWI on "god's coin toss".)

    I may have been too hasty there. Certainly, if there is a cluster of
many anthropic physical laws with about the same complexity, the above is
true. But there is likely to be only a few types of anthropic laws with
about the minimal complexity. In that case, a more flexible apparent set of
laws, like #2, is strongly favored.
    The most interesting possibility is that there may be a unique set of
physical constants leading to a mathematically consist theory of quantum
gravity - but, with the wrong values compared to what we see! Meaning that
some physical assumption, like diffeomorphism invariance, must be violated.
Why would that happen? Simple. Suppose there is an "interesting" set of
physical laws, of low complexity (thus with high symmetry), but which is not
anthropic. It might take only a little bit more complexity - a few extra
lines of code - to modify the algorithm of that "base case" into an
anthropic set of laws.
