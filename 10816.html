axiom Q3 of my ontic TOE. It says that any non null number admits a
successor.


Q1) Ax ~(0 = s(x))
Q2) AxAy ~(x = y) -> s(x) ­ s(y)
Q3) Ax ~(x = 0) -> Ey(x = s(y))

Together with the definition of addition:

Q4) Ax x + 0 = x
Q5) AxAy x + s(y) = s(x + y)

and the definition of multiplication:

Q6) Ax x * 0 = 0
Q7) AxAy x * s(y) = (x * y) + x
