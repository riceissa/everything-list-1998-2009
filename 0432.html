> Either strings can be concious, in
> which case consciousness is just as complicated as strings are, or only
> computations can be conscious, in which case consciousness is just as
> complicated as computations are.

It seems to me that to solve this problem we should look into the equivalence
between computation and information. When is a string a computation and when
is it information? It seems that the concepts of time and logical sequencing
is required for computation whereas information only requires logical self
consistency.
