I would like to point out that according to my somewhat more complicated
machine approach each configuration of each universe has more than one
valid successor depending on whether additional linkups occur between
print cycles or not. This is consistent with my longer text model. The
single long machine simplification will retain this character if a universe
has it.

I do not see any twinning at any such selection because given the theorem
cascade nature of the process postulated for our universe combined with
Chaitin's restriction on the complexity of a theorem which I posted on
earlier the less complex branch spawns a contradictory universe - a dead
trajectory.
