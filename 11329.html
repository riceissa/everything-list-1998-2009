I did put in parentheses "this of course assumes a robot can have experiences".
We can't know that this is so, but it seems a reasonable assumption to me. If we
had evolution with digital processors rather than biological processors do you think
it would have been possible for animals with similar behaviours to those with which
we are familiar to have developed? If so, do you think these animals would not
really have "experiences" despite behaving as if they did? Since evolution can only
work on behaviour, if zombie animals were possible why did we not evolve to be
zombie animals?
