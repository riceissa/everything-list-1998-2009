There is no such ordinal as "omega-1" in conventional mathematics. Keep in mind that ordinals are always defined as sets of previous ordinals, with 0 usually defined as the empty set {}...So,

0 = {}
1 = {0} = {{}}
2 = {0, 1} = {{}, {{}}}
3 = {0, 1, 2} = {{}, {{}}, {{}, {{}}}}

...and so forth. In thes terms, the ordinal "omega" is the set of finite ordinals, or:

omega = {0, 1, 2, 3, 4, ... } = too much trouble for me to write out in brackets

How would the set "omega-1" be defined? It doesn't make sense unless you believe in a "last finite ordinal", which of course a non-ultrafinitist will not believe in.
